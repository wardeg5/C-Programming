Arrays and Strings

This assignment focuses on using array structures and using string.h library. Strings are special case of Arrays

Instructions

Create a program called strings.c that does the following:

  1. Prompt the user and accept a string of no more than 254 characters
  2. Prompt the user and accept a second string of no more than 254 characters
  3. Display the length of both strings (not including the '\n' character)
  4. Create a third string consisting of the first half of string (1) and the second half of string (2).  Display this string
Your program must adhere to the class coding standards, and will be graded using the assignment grading criteria.  Submit your source code file to the Desire2Learn dropbox for this assignment.
